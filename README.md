# robintools

work in progress tools to make better trades on robinhood

```
mkdir ~/.robintools/
touch ~/.robintools/config.yml
```


example config file
```yml
account:
  email: "MYCOOLEMAIL"
  password: "MYSUPERSECRETPASSWORD"
```

### Tools

**capgains**  

Summarize your capital gains and losses

**crowbar**  

Find the right long call to buy
